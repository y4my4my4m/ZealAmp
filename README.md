# TOSamp

MP3 player for TempleOS

![TOSamp](https://git.checksum.fail/alec/TOSamp/raw/branch/master/screenshot.png "TOSamp")

# Requirements

- AC97 sound card

# Usage

Clone the repo, create a RedSea `ISO.C` image and add a `Music` folder, `#include "Run";`
